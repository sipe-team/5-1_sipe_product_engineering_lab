## 경쟁 분석 프레임

### 비교 관점
- 각 서비스가 풀고 있는 사용자 문제는 같은가? 다른가?
- 같은 문제를 풀되, 접근 방식이 어떻게 다른가?
- 각 서비스의 강점/약점은 무엇인가?

### 해자(Moat) 분석
- 네트워크 효과, 전환비용, 데이터 축적, 브랜드, 규모의 경제
- 참고: Hamilton Helmer의 "7 Powers"

## (선택) 본인 업무 역산 분석

2. 각각의 Why, KPI, ROI를 돌아보기
3. 가장 ROI가 낮았던 작업은? 왜?
